Carga de Datos a PostgreSQL

Laboratorio de agrocomputación y análisis epidemilógico

Esta aplicación esta diseñada para cargar datos desde Excel (archivos .xlsx) y hacer un preprocesamiento de los datos que contiene el archivos para la carga de estos a una base de datos en PostgreSQL alojada en el servidor del laboratorio.

Para poder consultar las bases de datos alojadas en el servidor se debe accesar a la ip 168.176.46.81 en el cual se asiganará una tabla con el nombre del archivo que se suba, por ende, este archivo no debe tener carácteres especiales (como la ñ o tíldes) ni espacios. Ej: nombre_datos.xlsx

Para acceder a la base de datos se debe tener un usuario y una contraseña, los cuales serán dados una vez se tengan las credenciales y aprovación por parte del encargado del laboratorio.
